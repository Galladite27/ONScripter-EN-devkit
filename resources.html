Engine

Engine download
The engine itself. If you don't know what you're doing, choose the "current" rather than "legacy" download. Latest version found online here.

Archive file tools
Tools for manipulating script & game asset archives. The usage of these tools is discussed as required in the guides; the tools also come with their own manual pages. The current version of the engine provided above comes with its own set of tools; these are legacy copies for just in case.

LUA API reference. Untranslated. LUA with *NScripter is not for the feint of heart. Not currently hosted online. Latest version found online here.
